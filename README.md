gotrim - Convenient line trimmer with sane defaults

## Quick start

    $ go get github.com/gregoryv/gotrim
    $ gotrim -h
    Usage: gotrim [OPTIONS]
    
    Options
        -c, --columns : 72
        -s, --suffix : "..."
        -t, --tab-width : 4
            number of spaces
    
        -h, --help
